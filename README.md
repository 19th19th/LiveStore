# ЛайвСтор

В сборке ЛайвСтор от liveopencart.ru убраны ненужные для РФ варианты оплат и доставок, загрузка данных с серверов opencartforum и opencart, исправлены некоторые ошибки и сделано множество доработок. Сравнение, список измнений и прочую информацию можно найти на [форуме](https://forum.liveopencart.ru/viewtopic.php?t=1268) или сайте [liveopencart.ru](https://liveopencart.ru/get).

Пожалуйста, прочитайте соответствующий файл (install.txt/upgrade.txt) внутри архива, или воспользуйтесь [статьей на сайте](https://liveopencart.ru/tips/obnovlenie-s-ocstore-3037-na-3039/) или смотрите [форум](https://forum.liveopencart.ru/).
